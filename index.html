Programme de Formation

Un parcours complet structuré en 4 chapitres pour maîtriser la stack MERN

Chapitre 01 : JavaScript (Fondamentaux)

  • Introduction et Environnement, Variables et types de données
  • Opérateurs et Conditions
  • Tableaux (arrays) et Objets
  • Chaines de caractères et Fonctions
  • Modules & JSON & stockage
  • Programmation asynchrone (introduction) et DOM (Document Object Model)
  • Gestion d'erreurs et debugging

Chapitre 02 : React

  • Introduction à React, JSX et l'environnement de développement
  • Composants, Props et Gestion de l'état (State)
  • Cycle de vie et Hooks React (useState, useEffect)
  • Gestion des événements et des formulaires
  • Rendu conditionnel et affichage de listes
  • Context API pour la gestion d'état globale
  • React Router pour la navigation côté client

Chapitre 03 : Node.js & Express.js

  • Mise en place d'un serveur Express
  • Routing avancé (GET, POST, PUT, DELETE)
  • Utilisation de Insomnia pour tester les API REST
  • Gestion des Middlewares pour le traitement des requêtes
  • Stratégies de gestion des erreurs

Chapitre 04 : MongoDB

Introduction à NoSQL et MongoDB

  • Concepts de base (Documents, Collections, BSON)
  • Installation et configuration de MongoDB
  • Utilisation de MongoDB Compass et du Shell pour les requêtes

Intégration avec Mongoose

  • Connexion à la base de données depuis Node.js
  • Définition de Schémas et de Modèles
  • Opérations CRUD (Create, Read, Update, Delete) complètes
  • Validation des données et Middlewares de Mongoose
  • Modélisation des relations entre documents
